About the Project

OdyssAI: A text adventure game led by an AI Dungeon Master.



Key Features
  • Browse tours with filtering and sorting
  • View detailed tour information including a map, reviews, and related tours
  • User accounts with authentication and authorization
  • Shopping cart to "book" tours
  • Payment processing with Stripe
  • Admin area to manage tours, users, reviews, etc.
Technologies Used
  • Node.js
  • Express
  • MongoDB
  • Pug templates
  • Stripe API
Note
  • Slow initial load time due to service provider's deployment hibernation during inactivity. Takes 20 Seconds to load
  • Not optimized for mobile
